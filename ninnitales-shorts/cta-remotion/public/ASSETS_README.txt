Drop your CTA assets in THIS folder, then reference the filenames in
src/CTA.tsx -> CTA_CONFIG.

What to add (all optional — it previews with gradients if empty):

  parent-away.jpg   Beat 1 ("the ache"): a parent away from home
                    (airport, hotel, working late). Vertical or it gets cropped.
  kid-bedtime.jpg   Beat 2 ("the magic"): a child in bed, warm phone/speaker glow.
  music.mp3         One song, reused on every video. Royalty-free recommended
                    (a baked-in copyrighted track can draw a YouTube Content ID
                    claim — harmless for a non-monetized brand channel, but
                    royalty-free avoids it entirely).

Sources that work in India: Pexels, Pixabay, Unsplash (free stock, no login),
or your own phone footage (best conversion). For music: Pixabay Music,
YouTube Audio Library, or Uppbeat (free with attribution).

Images are cover-fit to 1080x1920, so prefer vertical/portrait assets.
